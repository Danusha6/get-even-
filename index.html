What This Page Does

The purpose of the page is to provide a simple web portal with a form where users can input a number, and the page will determine if the input number is even or odd. Users can also click on the clear link to reset the form and enter a new number for evaluation.
Operation of the Page:
Form: The main content section contains a form with a single input field where users can enter a number. The form uses the "get" method and does not specify an action, so it will submit the form to the same page.
JavaScript: The page includes a JavaScript script at the end, which performs the following tasks:
The getEven(num) function takes a number as input and returns true if the number is even and false if it is odd. It checks the query string of the URL to see if there is any data submitted from the form. If data is submitted the script processes the input number, determines if it's even or odd using the getEven(num) function, and dynamically generates the output message which is the input and if it is even or odd.
